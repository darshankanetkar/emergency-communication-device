\# Working Principle



1\. User presses the emergency button.

2\. Arduino reads the button input.

3\. GPS module fetches real-time coordinates.

4\. GSM module:

&#x20;  - Sends SMS with Google Maps link

&#x20;  - Makes a call to predefined number

5\. System waits to prevent repeated alerts.



\# Features

\- One-click emergency alert

\- Real-time GPS tracking

\- SMS + Call functionality

\- Portable and battery-powered

\# 🚨 Arduino Emergency Communication Device



A portable one-click emergency alert system built using Arduino, GSM, and GPS modules.



\## 🔥 Features

\- Instant SMS with live GPS location

\- Automatic emergency call

\- Portable hardware design

\- Fast response system (\~40% quicker alerts)



\## 🛠 Tech Stack

\- Arduino (C++)

\- SIM900A GSM Module

\- NEO-6M GPS Module



\## 📍 How It Works

Press the button → GPS fetches location → SMS + Call sent instantly.

\## ⚙️ Setup

1\. Connect components as per circuit

2\. Upload code using Arduino IDE

3\. Insert SIM card in GSM module

4\. Power using battery
